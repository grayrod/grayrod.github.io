Numerical Extrema JS

Click Evaluate to calculate the minimum, maximum, area, and plot of the function on the given interval.
Results are generated using recursive numerical integration and differentiation algorithms implemented in JavaScript.

WTF is this?

A Mathematical Optimization algorithm created by Grayson Rodriguez that finds the minimum and maximum values (aka extrema) of any real-valued function that's continuous in a closed and bounded interval [a,b].   Basically, you give it a function and interval, and it gives you the function's minimum and maximum points (as well as the area under the function because I’m just that nice). You can also specify the degree of accuracy.
